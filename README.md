# GITHUB

it is version control to can work with team and edit project easily
## Installation

Run the following command to install Git:

```bash
sudo apt install git
```

## Usage

```python


# to push 
git push origin main

# to pull
git pull origin main

# to move to branch
git checkout branch_name
```
